## 为什么搭建博客

作为一名开发者，拥有一个个人博客是很有价值的。它不仅是记录学习笔记的工具，也是展示个人技术能力的名片。在这篇文章中，我记录了从零搭建博客的完整过程。

## 技术选型

我选择了纯 HTML/CSS/JS 方案，原因是：

- 无需构建工具，零依赖
- 完全可定制，不受主题限制
- 加载速度快，SEO 友好
- 可以部署在任何静态服务器上

## 博客结构

```
D:\blog\
  index.html          # 首页
  archives.html       # 归档页
  categories.html     # 分类页
  tags.html           # 标签页
  about.html          # 关于页
  css/
    style.css         # 全局样式
  js/
    posts-data.js     # 文章数据（由构建脚本自动生成）
    main.js           # 核心逻辑
  posts/
    article-1.html    # 文章详情页（由构建脚本自动生成）
    ...
  md-posts/           # ← 在这里放 .md 文件
    article-1.md
    article-2.md
    ...
  build.js            # 构建脚本
  build.bat           # 双击运行构建
```

## 功能清单

- 卡片式文章列表，带封面图和标签
- 暗色/亮色主题切换
- 全文搜索功能（Ctrl+K 快捷键）
- 按标签、分类筛选文章
- 时间线归档
- 响应式设计，移动端适配
- 分页导航

## 写文章的工作流

1. 在 `md-posts/` 目录下创建一个 `.md` 文件
2. 文件开头写 frontmatter（标题、日期、标签等）
3. 下面写正文（Markdown 格式）
4. 双击 `build.bat` 构建博客
5. 刷新浏览器查看效果

## 总结

> 搭建博客的目的是为了记录和分享。不要过度追求完美的主题和工具，先用起来、写起来，才是最重要的。
