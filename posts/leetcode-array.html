LeetCode 数组类题目解题思路总结

2026-04-15 算法 #LeetCode #算法

常用技巧

1. 双指针

适用于有序数组、原地操作等场景。经典题目：两数之和（排序后）、移除元素、合并有序数组。

// 例：移除有序数组中的重复元素
function removeDuplicates(nums) {
  let i = 0;
  for (let j = 1; j < nums.length; j++) {
    if (nums[j] !== nums[i]) {
      i++;
      nums[i] = nums[j];
    }
  }
  return i + 1;
}

2. 滑动窗口

适用于子数组/子串的最值问题。经典题目：最长无重复子串、最小覆盖子串。

// 例：最大子数组和（Kadane 算法）
function maxSubArray(nums) {
  let maxSum = nums[0], curSum = nums[0];
  for (let i = 1; i < nums.length; i++) {
    curSum = Math.max(nums[i], curSum + nums[i]);
    maxSum = Math.max(maxSum, curSum);
  }
  return maxSum;
}

3. 前缀和

适用于区间求和问题，可以将 O(n) 的查询优化到 O(1)。

// 构建前缀和数组
const prefix = [0];
for (let i = 0; i < nums.length; i++) {
  prefix[i + 1] = prefix[i] + nums[i];
}

// 查询区间 [l, r] 的和
function rangeSum(l, r) {
  return prefix[r + 1] - prefix[l];
}

总结

刷题不在多而在精。每做完一道题，思考它用了什么技巧，能否举一反三。总结套路比死记硬背更重要。
