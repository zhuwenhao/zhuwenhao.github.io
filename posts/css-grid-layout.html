CSS Grid 布局实战：从入门到精通

2026-05-05 前端开发 #CSS #前端

为什么需要 Grid

CSS Grid 是一种二维布局系统，可以同时控制行和列。与 Flexbox（一维布局）相比，Grid 更适合处理整体页面的布局结构。

基础概念

.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: 60px 1fr 40px;
  gap: 16px;
  min-height: 100vh;
}

上面的代码创建了一个经典的三栏布局：左 sidebar（200px）、中间内容区（自适应）、右 sidebar（200px），加上顶部导航栏和底部页脚。

常用属性

grid-template-columns / rows

/* 固定列宽 */
grid-template-columns: 200px 200px 200px;

/* 自适应列宽（等分） */
grid-template-columns: 1fr 1fr 1fr;

/* repeat 简写 */
grid-template-columns: repeat(3, 1fr);

/* 自动填充 */
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

grid-area 命名区域

.container {
  grid-template-areas:
    "header header header"
    "sidebar main aside"
    "footer footer footer";
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.aside   { grid-area: aside; }
.footer  { grid-area: footer; }

总结

CSS Grid + Flexbox 是现代前端布局的最佳组合。Grid 负责宏观布局（页面结构），Flexbox 负责微观布局（组件内部排列）。掌握这两个工具，几乎可以搞定所有布局需求。
