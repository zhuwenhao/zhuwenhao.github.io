Python 数据分析入门：Pandas 基础教程

2026-04-28 数据科学 #Python #数据分析

Pandas 简介

Pandas 是 Python 中最流行的数据分析库，提供了 DataFrame 和 Series 两种核心数据结构，让数据处理变得简单高效。

安装

pip install pandas

DataFrame 基本操作

import pandas as pd

# 从字典创建 DataFrame
data = {
    'name': ['Alice', 'Bob', 'Charlie'],
    'age': [25, 30, 35],
    'score': [85.5, 92.0, 78.5]
}
df = pd.DataFrame(data)

# 查看前几行
print(df.head())

# 查看基本信息
print(df.info())
print(df.describe())

# 选择列
print(df['name'])
print(df[['name', 'score']])

# 条件筛选
print(df[df['age'] > 28])

# 排序
print(df.sort_values('score', ascending=False))

数据清洗

# 处理缺失值
df.isnull()          # 检查缺失值
df.fillna(0)         # 填充缺失值
df.dropna()          # 删除缺失值行

# 去重
df.drop_duplicates()

# 类型转换
df['age'] = df['age'].astype(float)

分组聚合

# 按 category 分组并计算均值
result = df.groupby('category').agg({
    'score': ['mean', 'max', 'min'],
    'age': 'count'
})
print(result)

总结

Pandas 的学习曲线平缓，但功能非常强大。建议通过实际项目来学习，处理真实数据的过程中你会自然而然地掌握这些技能。
