VSCode 自定义代码片段提升开发效率

2026-05-10 开发工具 #VSCode #效率

什么是代码片段

VSCode 的代码片段（Snippets）功能允许你定义模板代码，通过输入简短的前缀触发键即可快速插入完整的代码结构。

如何创建自定义片段

  1. 打开命令面板：Ctrl+Shift+P
  2. 输入 "Preferences: Configure User Snippets"
  3. 选择语言（如 javascript.json）或创建全局片段
  4. 在打开的 JSON 文件中编写片段定义

片段语法详解

{
  "Console Log": {
    "prefix": "clg",
    "body": [
      "console.log('$1');",
      "$2"
    ],
    "description": "插入 console.log 语句"
  }
}

常用占位符

  • $1, $2 — 光标跳转位置（数字表示顺序）
  • ${1:default} — 带默认值的占位符
  • $TM_FILENAME — 当前文件名
  • $CURRENT_DATE — 当前日期

总结

善用代码片段，每天可以为你节省大量重复输入的时间。建议将日常开发中最常用的代码模式整理成片段。
