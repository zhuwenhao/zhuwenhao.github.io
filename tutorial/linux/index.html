# 防火墙

查看端口状态

sudo firewall-cmd --query-port=6666/tcp
1

添加端口

sudo firewall-cmd --add-port=6666/tcp --permanent
1

移除端口

sudo firewall-cmd --remove-port=6666/tcp --permanent
1

重新加载

sudo firewall-cmd --reload
1

sudo journalctl -u caddy -f -o json-pretty
