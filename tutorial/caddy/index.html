          # Caddy

          官网 (opens new window)

          # 下载

          打开 下载 (opens new window) 页面，平台选择 Linux amd64 ，点选 http.handlers.webhook 模块，然后点击下载按钮

          # 安装

          移动下载的二进制文件到 /usr/bin/

          $ sudo mv caddy_linux_amd64_custom /usr/bin/caddy
          1

          修改 caddy 权限

          $ sudo chmod 777 /usr/bin/caddy
          1

          测试是否有效

          $ caddy version
          1

          创建一个名为 caddy 的组

          $ sudo groupadd --system caddy
          1

          创建一个名为 caddy 且有可写的主目录的用户

          $ sudo useradd --system \
    --gid caddy \
    --create-home \
    --home-dir /var/lib/caddy \
    --shell /usr/sbin/nologin \
    --comment "Caddy web server" \
    caddy

          # 配置

          创建一个名为 Caddyfile 的配置文件

          $ sudo mkdir /etc/caddy
$ sudo touch /etc/caddy/Caddyfile
          1
          2

          配置文件内容

          example.com, www.example.com {
  root * /var/www/example.com
  file_server

  handle_errors {
    rewrite * /errors/{http.error.status_code}.html
    file_server
  }

  route /webhook {
    webhook {
      repo https://github.com/example/foobar.git
      path /var/www/example.com
      branch master
      secret xxx
    }
  }
}

          # 服务

          下载 (opens new window)服务文件并移动到 /etc/systemd/system/

          $ sudo mv caddy.service /etc/systemd/system/
          1

          启用 caddy 服务

          $ sudo systemctl daemon-reload
$ sudo systemctl enable --now caddy
          1
          2

          检查是否在运行中

          $ systemctl status caddy
          1
